### Python 2 -> Python 3 
```bash
# via 2to3 tool
# https://docs.python.org/2/library/2to3.html

# exmple : transform single python script 
$ 2to3 algorithm/python/bfs.py

# example : transform all script under file
$ 2to3 -w algorithm/python/*.py
```
